# Task 1

Було розроблено програму **«Система прогнозування успішності студентів»**.

Функціонал програми:
- завантаження даних з CSV-файлу;
- навчання моделі логістичної регресії;
- прогнозування результату (склав / не склав);
- перевірка введених даних;
- обробка помилок;
- збереження результатів у файл;
- ведення журналу роботи (`app.log`).

Програма має простий графічний інтерфейс та орієнтована на недосвідченого користувача.

Налаштування винесені у файл `config.properties`.
